## 项目简介
**BVE5-Motor-Assistance** 是一个专为 BVE Trainsim 5 开发的辅助工具，旨在帮助开发者管理和编辑列车电机音效项目。

**开发状态说明：**
目前仓库内包含的两个版本实现均处于**未完成**状态：
* **WPF 版本**：基于 Windows 平台的传统实现。
* **Avalonia 版本**：基于 Avalonia UI 的跨平台预览实现。

## 核心功能
* **音效项目管理**：支持定义音效项目名称及列车最大速度等核心参数。
* **配置文件处理**：内置了对 `settings.cfg` 文件的读取与修改逻辑。
* **多语言支持**：集成了中文（简体）、日语和英语的多语言资源。

## 技术栈
* **运行环境**：.NET 8.0
* **UI 框架**：Avalonia UI (v11.2.1) & WPF
* **架构模式**：MVVM (CommunityToolkit.Mvvm)

## 目录结构
* `BVE5-Motor-Assistance/`：WPF 版本源代码。
* `Avalonia-BVE5-Motor-Assistance/`：Avalonia 版本源代码。
